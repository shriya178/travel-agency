Jadoo - Travel Agency Landing Page ✈️🌍

A modern and responsive Travel Agency Landing Page built using HTML, CSS, and Bootstrap 5.
This project is inspired by the beautiful Jadoo UI Design and includes multiple responsive sections with clean layouts and minimal custom CSS.

🚀 Features
Responsive Navbar
Hero Section
Services / Category Section
Top Destinations
Booking Steps Section
Testimonials Carousel
Partner Brands Section
Subscribe CTA Section
Beautiful Footer
Bootstrap 5 Responsive Grid
Font Awesome & Bootstrap Icons
